# FakeDefender
本分支（web_dev）为本项目web服务器版本，其中机器学习部分只包括最基本的模型，模型优化（如人脸质量、角度等）的代码请push到main分支

## 如何运行
在项目根目录新建`/input`文件夹，把`/dfdc-pretrained-2`模型们放进去
1. `pip install -r requirement.txt`
2. `python main.py` （直接在shell运行，在pycharm中运行会有文件路径的不同）
